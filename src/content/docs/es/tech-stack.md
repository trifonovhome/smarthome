# Наш технологический стек

Мы не просто используем готовые облачные решения — мы строим свою инфраструктуру. Вот основные «киты», на которых она стоит:

1.  **OpenClaw**: Наш собственный Open Source ассистент. Он управляет знаниями, памятью и автоматизациями.
2.  **Home Assistant**: Стандарт индустрии для интеграции устройств. Мы используем его для управления светом, климатом и безопасностью.
3.  **Immich**: Лучшая self-hosted альтернатива Google Photos. Мы полностью контролируем наши семейные архивы.
4.  **GitHub**: Мы используем его как основу для версионирования нашей конфигурации и документации.

Главный принцип: всё, что можно запустить локально — должно работать локально.
