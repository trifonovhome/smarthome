# Pourquoi construisons-nous une maison intelligente Open Source ?

Bienvenue dans notre projet **SmartHome** !

Ce blog est l'histoire ouverte de la construction de notre écosystème domestique. Nous construisons un assistant familial qui est :
1. **Privé** : Toutes les données restent dans notre périmètre.
2. **Ouvert** : Nous partageons notre code et notre architecture.
3. **Familial** : Conçu pour le confort de chaque membre de notre famille.

Nous croyons que la technologie doit servir les gens, et non l'inverse. Vous trouverez ici des descriptions de nos erreurs, de nos succès et des détails techniques sur la façon dont nous avons automatisé notre maison.

Bienvenue dans notre voyage !
