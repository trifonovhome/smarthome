# Notre matériel : Inventaire du matériel

Pour construire une maison intelligente fiable, vous avez besoin d'une base stable. Voici le matériel sur lequel repose notre projet :

*   **Hôte OpenClaw (esh.local)** : Mini PC GEEKOM IT13S (Intel i9-13900HK, 32 Go de RAM). C'est le « cerveau » de notre assistant, il traite toutes les requêtes, gère la logique et stocke les sauvegardes.
*   **Home Assistant (homeassistant.local)** : NiPoGi Essenx E1 (AMD Ryzen 3250U, 8 Go de RAM). Gère la communication avec les appareils physiques de la maison.
*   **Bureau familial** : iMac (24", M1). L'ordinateur principal de la famille.
*   **Routeur** : Keenetic. Notre gestionnaire central du trafic réseau.

Nous avons sélectionné ces appareils en nous basant sur le principe d'une faible consommation d'énergie + hautes performances + fiabilité.
