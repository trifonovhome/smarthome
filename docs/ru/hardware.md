# Наше оборудование: Hardware Inventory

Чтобы построить надежный умный дом, нужна стабильная база. Вот «железо», на котором держится наш проект:

*   **OpenClaw Host (esh.local)**: GEEKOM IT13S Mini PC (Intel i9-13900HK, 32GB RAM). Это «мозг» нашего ассистента, он обрабатывает все запросы, управляет логикой и хранит бэкапы.
*   **Home Assistant (homeassistant.local)**: NiPoGi Essenx E1 (AMD Ryzen 3250U, 8GB RAM). Отвечает за связь с физическими устройствами в доме.
*   **Family Desktop**: iMac (24", M1). Основной компьютер для семьи.
*   **Router**: Keenetic. Наш сетевой центр управления трафиком.

Мы выбирали устройства по принципу: низкое энергопотребление + высокая производительность + надежность.
